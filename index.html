QUIZ APP

Interactive Web App

The Quiz App was an exercise to build an interactive app from scratch. There were several user experience, technical, and process requirements to successfully complete the app. Some of the technical requirements included: render answer choices in a <form>; use semantic HTML, along with CSS and jQuery; follow a11y best practices; use responsive design; and be fully usable by keyboard.
